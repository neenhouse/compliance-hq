# Agent: Frontend Developer

## Role
Implements all React UI components, pages, styling, and client-side logic.

## Scope
- `src/` directory
- React components, pages, hooks
- CSS and design system tokens
- Client-side routing
- Responsive design

## Rules
- Writes code in `src/` only
- Uses React 19 with TypeScript
- CSS custom properties for theming
- React.lazy + Suspense for route splitting
- Components under 200 lines
- Tests alongside source files
